Requisitos Funcionais (RF)
Código	Descrição
RF01	O sistema deve permitir a consulta de empresas por meio do CNPJ.

RF02	O sistema deve exibir os dados cadastrais da empresa consultada.

RF03	O sistema deve listar os sócios e administradores (QSA) da empresa.

RF04	O sistema deve identificar empresas relacionadas por meio dos sócios vinculados.

RF05	O sistema deve permitir a busca de empresas por nome de sócio.

RF06	O sistema deve identificar automaticamente grupos econômicos com base nos vínculos societários.

RF07	O sistema deve gerar um relatório básico de análise de risco de crédito da empresa.
