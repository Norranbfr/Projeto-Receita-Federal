
Requisitos Não Funcionais (RNF)

Código	Descrição

RNF01	O sistema deve ter tempo de resposta de até 3 segundos para operações comuns.

RNF02	A interface deve ser intuitiva, simples e de fácil utilização para o usuário final.

RNF03	O sistema deve garantir segurança através de controle de acesso, autenticação e proteção de dados.

RNF04	O sistema deve possuir alta disponibilidade, minimizando indisponibilidades e falhas.

RNF05	O sistema deve ser escalável, suportando aumento no número de usuários sem perda de desempenho.

RNF06	A arquitetura do sistema deve ser baseada em API REST para comunicação entre serviços.
