Diagrama do Banco de Dados

Este diagrama representa a estrutura do banco de dados do sistema.

## Entidades

### Empresa
Armazena os dados das empresas consultadas.

### Sócio
Armazena os dados dos sócios e administradores.

### Participação
Responsável pela ligação entre empresas e sócios.

## Objetivo

O modelo foi desenvolvido para identificar vínculos entre empresas e auxiliar na análise de crédito.
