## 🎯 Objetivo
 
Desenvolver um sistema que auxilie na **análise de risco de crédito**, permitindo:
 
- Consultar empresas pelo CNPJ
- Visualizar o Quadro de Sócios e Administradores (QSA)
- Identificar vínculos societários entre entidades
---
 
## 🛠️ Tecnologias Utilizadas
 
- **Python**
- **PostgreSQL**
---
 
## ⚙️ Funcionalidades
 
- Consulta de empresas por meio do CNPJ
- Geração de relatório básico de análise de risco de crédito
- Busca de empresas por nome de sócio
---
 
## 🧭 Escopo do Sistema
 
### ✅ Dentro do escopo
 
- Consulta de CNPJ
- Exibição de dados da empresa
- Análise de sócios (QSA)
- Identificação de vínculos societários
- Relatório básico de risco
### 🚫 Fora do escopo
 
- Busca por CPF
- Cadastro manual de empresas
- Alteração de dados cadastrais
- Integração com sistemas bancários reais
