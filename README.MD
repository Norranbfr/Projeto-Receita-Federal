# 🎯 Objetivo
Desenvolver um sistema que auxilie na análise de risco de crédito, permitindo
consultar empresas pelo CNPJ, visualizar o Quadro de Sócios e Administradores (QSA)
e identificar vínculos societários.

# 🛠 Tecnologia Utilizadas

## ⚙ Funcionalidades
- O sistema deve permitir a consulta de empresas por meio do CNPJ.
- O sistema deve gerar um relatório básico de análise de risco de crédito da empresa.
- O sistema deve permitir a busca de empresas por nome de sócio.
